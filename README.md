
## Runshare



### 環境構築

Server Side: Firebase</br>
Frontend: React(Redux)</br>
Docker

イメージをビルド
`docker-compose build`

依存ライブラリのインストール
`docker-compose run —-rm node npm install`

 コンテナ起動
 `docker-compose up`
 
 コンテナ停止
 `docker-compose down`

**Note: this is a one-way operation. Once you `eject`, you can’t go back!**
